# PTGDUD

Đây là repository tổng hợp đề bài và lời giải cho các bài thực hành, bài lab và bài tập môn **Phát triển Giao diện Ứng dụng (PTGDUD)**.

## Cấu trúc repository

- `Labs`: chứa các file đề bài
- `BaiTap`: chứa phần bài làm tương ứng với từng đề

Các thư mục con trong `BaiTap` được đặt tên trùng hoặc tương ứng với tên các file đề trong `Labs`, giúp dễ dàng đối chiếu nội dung giữa đề bài và lời giải.

## Mục đích

Repository này được xây dựng nhằm:
- Lưu trữ bài tập một cách có hệ thống
- Dễ dàng tra cứu giữa đề bài và phần lời giải
- Hỗ trợ học tập, thực hành và ôn tập trong quá trình học môn PTGDUD
- Thuận tiện cho việc cập nhật, chỉnh sửa và quản lý mã nguồn trên GitHub
